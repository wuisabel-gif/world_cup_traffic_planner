# Contributing to World Cup Traffic Planner 🚇⚽

Thanks for stopping by.

This project exists for one reason:

> Help people survive World Cup traffic around SoFi Stadium.

Whether you're improving transit routes, fixing bugs, polishing the UI, or making the planner smarter, contributions are welcome.

---

## Ways to Contribute

### 🚉 Improve Transit Planning

Ideas include:

- Better Metro routing
- Additional stations
- Improved transfer guidance
- Accessibility considerations
- Shuttle route updates

### 🚗 Improve Traffic Planning

Ideas include:

- Better congestion estimates
- Additional freeway information
- Smarter disruption windows
- Alternative routing suggestions

### 🗺️ Improve the Map

Ideas include:

- Better route rendering
- Mobile improvements
- Additional landmarks
- Visual polish

### 🎨 Improve the UI

Ideas include:

- Accessibility improvements
- Better mobile layouts
- Dark mode enhancements
- Improved visual hierarchy

### 🐛 Fix Bugs

Found something broken?

Open an issue or submit a fix.
